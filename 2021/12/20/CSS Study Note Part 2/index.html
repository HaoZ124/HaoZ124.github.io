CSS font attribute

CSS font attributes are used to define the font family, size, weight, and text style (such as italics)

Font-family

  • CSS uses the font-family attribute to set the font family of the text
    div {font-family:Arial,"Microsoft Yahe","Microsoft Yahe";}
  • The various fonts must be separated by commas
  • In general, if there is a font composed of multiple words separated by spaces, add quotation marks
  • The most common fonts:

    Font-size

    CSS uses the font-size attribute to set the font size
  • px (pixel) size is the most commonly used unit of our web pages
  • The default text size of Google Chrome is 16px

    Font-weight

  • CSS uses the font-weight attribute to set the font weight
    font-weight: normal | bold | bolder | lighter |number
    Attribute Description
    Normal default value (not bold), equivalent to number=400
    Bold defines bold (bold), equivalent to number=700
    Bolder defines special bold
    100-900 400 is equivalent to normal, 700 is equivalent to bold, note that this number is not followed by a unit

In development environment, using numbers to indicate thickness is preferred.

Font-style

  • CSS uses font-style attribute to set text style
    Attribute Usage
    Normal The default value, the browser will display the standard font style
    Italic browser will display the italic font style

Note: We seldom italicize the text in general. Instead, we need to change the italic label (em, i) to a non-italic font

Compound writing of font properties

The font compound attribute can be written by combining the above text styles, so that writing code is more economical, for example:

  • Order: text style, text thickness, text size, text font
  • When using the font attribute, it must be written in the order in the above grammatical format, the order cannot be changed, and the attibutes are separated by spaces
  • The properties that do not need to be set can be omitted (take the default value), but the font-size and font-family properties must be retained, otherwise the font attribute will not work

    Summary of font attibutes

    Attribute Meaning Notice
    font-style Font style Remember that the tilt is italic, and the non-tilt is normal. Normal is the most commonly used in actual development
    font-weight font weight is 700 or bold for bold, normal or 400 for non-bold, remember that the number does not add a unit
    font-size font number we usually use is px, we must use with unit
    font-family font
    Font ligation Font ligation is in order and cannot be changed position arbitrarily. Font and font size must appear at the same time
