CSS Label Selector

Label Selector

  • The role of the label selector is to specify a unified CSS style for a certain type of label on the page
  • The benfit of label selector is to quickly set the style for the same type of label on the page.
  • The disadvantage is that you cannot design a differentiated style, you can only select all the current labels

Specification:

  • Before the property value, after the colon, leave a space
  • Leave spaces between label names and braces

Class Selector

If you want to choose different labels differently, select one or several labels individually, you can use the class selector

The structure needs to be called with the class property
<div class='red'> turns red </div>

Notice:

  • The class selector is represented by the class property in HTML, and in CSS, the class selector is represented by a dot.
  • Style point definition, structure class call

Class Selector with Multiple Class Name

  • We can assign multiple class names to a label, the simple understanding is that a label has multiple names
    <div class="red font20">Lin Xiao</div>
  • Write multiple class names in the label class property
  • Multiple class names must be separated by spaces
  • This label can have the styles of these class names respectively

Scenarios used in the development of multiple class names:

  • You can put the same style (common part) of some label elements into a class
  • These labels can call this public class, and then call their own unique class
  • So that can make code concise, and easy to be modified together

ID Selector

  • The id selector can specify a specific style for HTML elements marked with a specific id
  • HTML elements use id propertys to set id selectors, and in CSS, id selectors are defined by “#”

Notice:

  • The id property can only appear once in each HTML document.
  • Memory formula: style # definition, structure id call. Can only be called once, others should not use

The Difference

What is the difference between id selector and class selector?

  • The class selector (class) is like a person’s name, a person can have multiple names, and a name can be used by multiple people at the same time
  • The id selector is like a person’s ID number, which is unique in China and must not be repeated.
  • The biggest difference between id selector and class selector is the number of uses
  • The class selector is the most used to modify the style. The id selector is generally used for the unique elements of the page, and is often used in conjunction with JavaScript.

CSS Universal Selector

In CSS, universal selector are defined by *, which means to select all elements (labels) in the page.

The universal selector does not need to be called, it automatically styles all elements
Generally used to clarify the inner and outer margins of the element label

Summary

|Basic selector |Function |Features |Frequency |Grammar|
|:-:|:-:|:-:|:-:|
|Label selector |can select all the same labels |can not differentiate the selection |often | p {color: red}|
|Class selector |can select one or more labels |choose a lot according to needs |very often |.nav {color: red}|
|ID selector |can only select 1 label at a time |ID property can only appear once in each HTML document|work with js |#nav {color: red}|
|Universal selector |Select all the labels |too many choices than needs |used in special case |* {color: red}|
